# Digital Studio

Website landing page untuk **Digital Studio**, sebuah usaha rintisan mahasiswa yang bergerak di bidang jasa pembuatan website dan branding digital khusus untuk UMKM dan bisnis mahasiswa.

## Tentang Proyek

Website ini dirancang sebagai halaman representasi bisnis yang fokus pada:
- Profil usaha dan value proposition
- Layanan yang ditawarkan (pembuatan website, SEO, branding, landing page)
- Keunggulan kompetitif
- Call-to-action untuk konversi

## Fitur SEO

### Meta Tags
- `charset` UTF-8
- `viewport` untuk responsif
- `title` dengan keyword utama
- `meta description` deskriptif
- `meta keywords` relevan
- `meta author`
- `meta robots` index, follow
- Open Graph tags untuk social sharing
- Canonical URL

### Struktur Heading
- Satu H1 di Hero Section
- H2 untuk setiap section utama
- H3 untuk sub-item

### Schema Markup
Website menggunakan Schema.org JSON-LD untuk Organization:

```json
{
    "@context": "https://schema.org",
    "@type": "Organization",
    "name": "Digital Studio",
    "url": "https://digimahasiswa.github.io/",
    "description": "Usaha rintisan mahasiswa...",
    "founder": {...},
    "contactPoint": {...}
}
```

## Kesiapan SEM

Website sudah siap digunakan sebagai landing page Google Ads:
- Headline dengan keyword utama
- Value proposition yang jelas
- CTA (WhatsApp dan Email)
- Trust signals berupa statistik
- Desain mobile-first
